Convenience functions for setting up the HTTP interface of Riak.

Description

Convenience functions for setting up the HTTP interface of Riak. This module loads parameters from the application environment:

web_ip
IP address that the Webmachine node should listen to
web_port
port that the Webmachine node should listen to
web_logdir
directory under which the access log will be stored
